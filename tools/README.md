# Tools

This directory contains custom tools and utilities developed as part of my cybersecurity learning journey.

The goal of these projects is to understand how common security workflows operate internally while improving my programming skills through practical applications.

## Objectives

* Learn Python through hands-on projects.
* Build reusable tools instead of one-off scripts.
* Explore networking, reconnaissance, enumeration, and automation concepts.
* Understand the underlying principles behind existing security tools.

## Current Projects

### Python Port Scanner

A simple multi-threaded TCP connect scanner written in Python.

Current capabilities:

* Scans ports 1–1024.
* Uses Python sockets for TCP service discovery.
* Uses threading to improve scan speed.
* Collects and displays open ports after completion.

## Future Additions

As this repository grows, this directory may include:

* Service detection modules.
* Banner grabbing utilities.
* Reconnaissance automation scripts.
* Parsing and reporting tools.
* Wrappers around common Linux commands.
* Data processing and result aggregation utilities.
* Additional networking and cybersecurity projects.

## Philosophy

These projects prioritize learning and understanding over recreating existing tools. The objective is to develop practical skills by building components incrementally and improving them over time.

## Disclaimer

These tools are developed for educational purposes and are intended to be used only on systems for which explicit authorization has been granted, such as personal lab environments, CTF platforms, and intentionally vulnerable machines.
